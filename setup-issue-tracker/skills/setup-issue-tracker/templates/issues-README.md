# Issues

Durable local implementation issues live in this directory as `NN-slug.md` files. The numeric prefix is the ticket's permanent id and the path is a stable permalink: reference tickets by `docs/issues/NN-slug.md`, and never move or rename these files.

Rules:

- Store durable implementation tickets here, not in temporary workspace.
- Include a `Status:` line near the top of each issue, using a value from `docs/agents/triage-labels.md`.
- Keep issues independently implementable when possible.

Progression through phases is tracked on the status board under `docs/issue-tracker/`; see `docs/agents/issue-tracker.md`.
